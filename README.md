# Target Audience Identification using Data Science

## 📌 Project Overview
This project focuses on identifying target audiences using machine learning techniques.

## 🔧 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## 📊 Methods Used
- K-Means Clustering for segmentation
- Random Forest for prediction

## 📈 Output
- Customer segments based on income and spending behavior
- Prediction of new customer category

## 🎯 Conclusion
This project helps businesses understand customer behavior and improve marketing strategies.
